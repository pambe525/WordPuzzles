Release Notes

v0.8: [3/15/2022]

All Puzzles List
  • Shows all published puzzles in groups of 10 per page
  • Puzzles list can be sorted by various puzzle properties, in ascending or descending order.
  • Puzzlers can click on the puzzle title to solve or preview the puzzle.
  • This list can be accessed by the SHOW ALL PUZZLES button on Dashboard.

v0.7: [3/12/2022]

Dashboard
  • Dashboard is the home page and shows Recently Published puzzles and user's Draft Puzzles.
  • Puzzlers can see puzzles published within the last 7 days in the Recently Published section.
  • Puzzle editor can preview and unpublish a puzzle by clicking on the puzzle title.
  • Puzzlers can see their unpublished draft puzzles in the Draft Puzzles section, last edited first.
  • Puzzle editors can edit their puzzle by clicking on the title of the puzzle in the draft section.
  • Each puzzle in the draft list offers icon buttons to preview or delete the puzzle.

v0.5: [2/28/2022]

Word Puzzle Management
  • Puzzler can create a New Puzzle (crytic or non-cryptic clues).
  • Word puzzles can be edited and saved.
  • Puzzle editor can Add Clue(s) to the puzzle.
  • Points and parsing note can also be added to each clue.
  • Puzzle editor can delete the puzzle (after confirmation).
  • Puzzle editor can Preview and Publish the puzzle (after clues are created).
  • Once published, a puzzle is available to all logged in puzzlers to solve. Published puzzles cannot be edited.
  • Unpublish a puzzle to withdraw it from availibility to other puzzlers or to edit errors.

v0.3: [2/10/2022]

User Authentication
  • New User can create an account with username, password and email address.
  • Existing users can login with username and password.
  • Reset Password if user has forgotten the password.
  • Once signed in, user can edit their Account Settings including username, email, first and last name.
  • Users can Change Password.
