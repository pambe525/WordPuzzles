Release Notes
User Authentication [2023.05.09]
  • New users can Sign Up using a username, password and email address. The username must be unique and is not case-sensitive. This will be visible to others, so use your first name or full name as username (no spaces). The email address must also be unique and will be used for notifications.
  • Registered users can Log In with their username and password. On successful login, users are directed to the Dashboard page.
  • Users can Reset Password if they forgot the password. An email is sent with a link which they can use to reset the password.
  • Once signed in, users can edit their Account Settings including username, email, first and last name.
  • Users can also Change Password, if needed.
Responsive Web Interface [2023.05.09]
  • The web interface is responsive and adapts to the device's viewport such as large monitors, laptops, tablets and phones.
  • On larger devices, a side menu is displayed that provides access to all the features.
  • On smaller devices, the side menu can be accessed using the icon on top right of the screen.
  • Clicking on the CrypticLive logo always navigates to the home page (Dashboard)
