Release Notes
[July 13, 2023]
Dashboard Notifications
  • My Notifications section on the Dashboard (home page) displays pertinent links and notifications.
  • If you have unfinished puzzles to solve, a notification will link to the puzzles list to display unfinished puzzles. It is recommended that you complete the unfinished puzzle by solving or revealing clues.
Puzzle Session Scores (Individual Sessions)
  • In the list of Published Puzzles, any puzzle that has been attempted or solved by other users will have a [] icon button which can be clicked to display scores.
  • Scores are listed from highest to lowest by user. A icon next to the score indicates that the puzzle was completed by the respective user, whereas a icon indicates an incomplete session.
  • The number of clues solved and revealed are also displayed for each user session.
Solving a Puzzle (Individual Session)
  • A published puzzle (on the Published Puzzles page) can be solved by clicking on the title. This will open the Puzzle Session page. if you wish to start an individual solve session click on the [Start Individual Solve Session] button. This will display all the clues and session status.
  • Each clue can be solved by clicking on the clue text. Enter your answer in the dialog that pops up. If answer is incorrect an error message will be displayed. Correct answer input will close the dialog and update the clue status in the list of clues. The score is also updated in the session panel.
  • You can also reveal a clue by clicking on the icon. This will reveal the answer (no points are earned on revealed answers).
  • When all clues are solved (or revealed) the session status will be shown as Complete and the final score will be displayed.
  • You can close the Puzzle Session page anytime by clicking on the button on top right. The session can be resumed by clicking on the puzzle title on the Published Puzzles page.
Published Puzzles List
  • The Published Puzzles page lists all the published puzzles. This page can be accessed from the side menu. The list contains details for each puzzle. Clicking on the title of the puzzle redirects to the Solve Puzzle page, if you are not the editor of the puzzle. Puzzle editors are directed to the Edit Puzzle page where the puzzle can be unpublished or edited in a limited way.
  • Status icons are displayed on each puzzle badge to indicate session states:
    indicates that you completed the individual puzzle session.

    ME in the time log indicates that you are the editor of the puzzle.
  • Each published puzzle that has other user sessions, will have the [] icon button displayed. This can be clicked to display scores for all the individual sessions associated with the puzzle.
Publish a Puzzle
  • Publishing a puzzle makes it accessible to all users to solve. Review all the clues before you publish. Once published, no new clues can be added. To re-edit the puzzle or add new clues unpublish the puzzle first, which puts it back in draft mode.
  • [Publish]: This button appears on the Edit Puzzle page once at least 5 clues have been defined. A confirmation dialog will appear before the puzzle is posted. Once published, the puzzle will appear on the Published Puzzles page.
  • [Unpublish]: This button is accessible on the Edit Puzzle page. When clicked, the puzzle will be unpublished and put back in draft mode for editing. This button appears only if other users have not yet attempted to solve the puzzle in an individual session.
Creating Draft Puzzles
  • [Create a New Puzzle]: Click on this link ion the dashboard to create a new puzzle. Select the type of puzzle, add a description and click Create Puzzle to start editing the puzzle and adding clues. The new puzzle will appear in your drafts list.
  • My Draft Puzzles section: This section appears on the Dashboard when one or more draft puzzles are created. Click on the title of the puzzle in the list, to edit it in the Puzzle Editor. Click the button to delete the puzzle. Note that all clues associated with the puzzle will also be deleted. The puzzle badge lists last edited timestamp in local time.
Puzzle Editor
  • Clicking on the puzzle link in the draft puzzles section on the home page will open the edit puzzle page. Here you can edit the description (click on the [Edit] button). You can also add bulk clues by clicking on the [Add Clues] button.
  • [Add Clues] This opens the add clues form. Multiple clues and corresponding answers can be entered or cut & pasted. Each item in the text boxes must be separated with a new line [return] and must begin with the clue number.

    Note that clue text need not have answer length specified in parenthesis since these will be auto-added based on the answer given for each clue. Answers must be unique across clues and cannot contain non-alphabet characters (only spaces and hyphens). Cryptic and non-cryptic clues can only have a single answer.

    Existing clues are replaced if re-entered. Numbering gaps should be managed by the puzzle editor.
  • Clues List: Clues are listed in order of clue numbers on the edit puzzle page. Click on the clue description to edit the clue (see below). You can also delete the clue by clicking on the button.
  • Edit Clue Form: This form allows you to make changes to a single clue. Changed fields will overwrite the existing clue on submit. Note that answers will automatically uoper-cased. You can also add parsing or change the number of points for each clue using this form.
User Authentication
  • [Sign Up]: New users can register using a username, password and email address. The username must be unique and is not case-sensitive. This will be visible to others, so use your first name or full name as username (no spaces). The email address must also be unique and will be used for notifications.
  • [Log In]: Registered users can sign in with their username and password. On successful login, users are directed to the Dashboard page. Site pages are not accessible unless the user is authenticated.
  • [Reset Password]: Users can reset their password if they forget it. An email is sent with a link which they can use to reset the password.
  • [Account Settings]: Once signed in, users can edit their credentials including username, email, first and last name. This can be accessed from the side menu.
  • [Change Password]: Users can also change their password, if needed. This can be accessed from the side menu.
  • [Logout]: Users can logout (accessible from the side menu). This will take them back to login page.
Responsive Web Interface
  • The web interface is responsive and adapts to the device's viewport such as large monitors, laptops, tablets and phones.
  • On larger devices, a side menu is displayed that provides access to all the features.
  • On smaller devices, the side menu can be accessed using the icon on top right of the screen.
  • Clicking on the CrypticLive logo always navigates to the home page (Dashboard)
